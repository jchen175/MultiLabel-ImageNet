# CutLER - Modified Implementation

Third-party implementation: https://github.com/facebookresearch/CutLER

## Modifications

### `maskcut/maskcut_dinov3.py`  
Extends implementation to support DINOv2/v3 models for pseudo-mask generation.

### `maskcut/merge_scattered_jsons.py`
Merges split job results into a final output file with maskcut object proposals in COCO format.
